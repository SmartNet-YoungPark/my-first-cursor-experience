**Tetris**  
A simple, modern Tetris-style game you can play in the browser.

---

## Features

- **Keyboard controls:**
  - Move: <kbd>←</kbd> <kbd>→</kbd>
  - Rotate: <kbd>↑</kbd>
  - Soft drop: <kbd>↓</kbd>
  - Hard drop: <kbd>Space</kbd>
  - Pause / resume: <kbd>P</kbd>
- Score, level, and line count
- Next-piece preview
- Clean UI and responsive layout

---

## Getting started

### 1. How to run

1. Clone this repository or download the files.
2. Open `index.html` by double-clicking it or opening it in your browser.
   - No server or build step is required.

### 2. File layout

```
├── index.html       # Main HTML
├── style.css        # Styles
└── game.js          # Game logic (JavaScript)
```

## Controls

- **← / →** — Move the piece left or right  
- **↑** — Rotate  
- **↓** — Soft drop (fall faster)  
- **Space** — Hard drop (instant drop to the bottom)  
- **P** — Pause / resume
